真島がAIと歩んだ記録

2026.04.15

AIと一緒にゼロからサイトを作り、Gitの流れをひと通り学んだ

Claude Code（AI）との会話だけで、このサイト全体を構築。コードを一行も自分で書かず、指示を出すだけでページが完成していく体験は新鮮だった。

  • index.html・ai-log.html・notes.html・記事ページをAIで作成
  • デザイン（背景グラデーション・ボタン・カードUI）もすべてAIに指示して調整
  • GitHub に mashima-sandbox リポジトリを新規作成してプッシュ
  • main / staging のブランチ運用を実践（切り替え・同期・マージ）
  • Pull → Commit → Push → Merge の4ステップの流れを体で覚えた
  • 学んだ内容をノートページにまとめ、サイト自体が学習記録になった
